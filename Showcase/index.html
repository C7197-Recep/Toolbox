About Showcase

"Coder Showcase" App :

Showcase makes navigation of Github repos easier.

This interface pulls repos or subfolders of a repo defined by a Github user, and shows them as instantly navigable tabs.

So, you can use Showcase to show your projects under Github to your potential customers or to the company you want to apply.

Usage :

Just pass your Github username and name of the repo you want in showcase and navigate to the url below.

Example :

https://c7197-recep.github.io/Toolbox/Showcase/index.html?user=C7197-Recep&repo=Clarusway

Parameters :

Defining the user : user=GithubUsername

For all repos : repo=all

For subfolder of a repo : repo=TheRepoName

To See All Files : showfiles=true
